HTB: Introduction to Active Directory 

This repository contains my documentation, lab notes, and practical exercises from the [Introduction to Active Directory](https://academy.hackthebox.com/app/module/74) module on Hack The Box Academy.

Module Summary
Active Directory (AD) is the backbone of most corporate environments. This module focuses on the core concepts needed to enumerate and attack AD environments, covering functionality, schema, and the protocols used for authentication, authorization, and accounting (AAA).

Learning Objectives
- **AD Structures:** Examine the history and hierarchy of AD objects (Users, Groups, Computers).
- **Protocols:** Deep dive into authentication protocols like **Kerberos**, **NTLM**, and **LDAP**.
- **Security:** Understand the difference between **Rights** and **Privileges**.
- **Management:** Practice common AD administrative tasks and identify common misconfigurations.
