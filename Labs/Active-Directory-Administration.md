# Lab: Active Directory Administration & Governance

## Preface
In this lab provided by [Hack The Box Academy](https://academy.hackthebox.com/), I was presented with a challenge that required me to conduct standard Domain Administration procedures. Following the style of industry write-ups, this report documents the lifecycle of identity management and policy enforcement within a simulated corporate network.

---

## Overview
The objective was to manage the `inlanefreight.local` domain, acting as a Domain Administrator. This involved handling the "day-to-day" security and operational tasks that keep an enterprise network running smoothly and securely.

### 🛠 Environment & Tools
* **Target Domain:** `inlanefreight.local`
* **Primary Interface:** Windows Server 2019 / 2022
* **Management Tools:** Active Directory Users and Computers (ADUC), Group Policy Management Console (GPMC), and DNS Manager.
